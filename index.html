String.remove()

Remove all instances of inputted arguments. Takes in 2 to infinity arguments. The first argument is what gets put in the spot of the removed element.

HOW TO USE: variable = variable.remove('string','string')

Example setup: mystring = 'hello_all#remove_&stuff'. Example use: myString.remove('','_','&','#'). Example return: returns helloallremovestuff
